# Equivariant Graph Attention Network - 3D Conformers & Feature Fusion (EGAN-3DF)
This code implements a hybrid deep learning framework for binding affinity prediction that integrates 3D ligand information using Equivariant Graph Neural Networks (EGNNs) across multiple conformers. An attention mechanism is used to weight contributions from different conformers, and traditional molecular descriptors are incorporated through feature fusion to enhance representation. The model is implemented in Python using PyTorch and demonstrates improved predictive performance over existing ligand-only approaches on benchmark datasets.
